# [PATCH] 이메일 OTP 발송 수정 — Antigravity(FastAPI 비동기) 환경 전용

> **근본 원인:** FastAPI는 async 기반 서버인데, 기존 코드의 `smtplib`(동기)를
> async 라우터 안에서 직접 호출하면 이벤트 루프가 블로킹되어 요청이 멈추거나 실패함.  
> **해결책:** `aiosmtplib`(비동기 SMTP 라이브러리)로 전면 교체.  
> **수정 파일:** `requirements.txt`, `.env`, `otp_service.py`, `routers/auth.py`

---

## 3. `app/services/otp_service.py` — `send_otp_email` 함수 전체 교체

> `generate_and_store_otp`, `verify_otp` 등 다른 함수는 수정하지 않는다.  
> `send_otp_email` 함수만 아래 코드로 전체 교체한다.

**변경 전 (기존 동기 코드 — 삭제)**

```python
# ❌ 삭제 대상: smtplib 기반 동기 함수 전체
def send_otp_email(email: str, otp: str, name: str):
    ...
    with smtplib.SMTP_SSL(...) as server:
        ...
```

**변경 후 (비동기 교체 — 적용)**

```python
# ✅ app/services/otp_service.py

import ssl
import logging
import aiosmtplib
from email.mime.text import MIMEText
from email.mime.multipart import MIMEMultipart
from app.core.config import settings

logger = logging.getLogger(__name__)


async def send_otp_email(email: str, otp: str, name: str) -> None:
    """
    비동기 OTP 메일 발송 — HiWorks SMTP (SSL/465) 전용.
    FastAPI async 라우터에서 await으로 호출해야 한다.
    """
    subject = "[인증] 이메일 인증 코드 안내"
    body = (
        f"안녕하세요, {name}님.\n\n"
        f"인증 코드: {otp}\n\n"
        f"본 코드는 5분간 유효합니다.\n"
        f"본인이 요청하지 않은 경우 이 메일을 무시하세요."
    )

    msg = MIMEMultipart("alternative")
    msg["Subject"] = subject
    msg["From"]    = settings.SMTP_FROM
    msg["To"]      = email
    msg.attach(MIMEText(body, "plain", "utf-8"))

    try:
        await aiosmtplib.send(
            msg,
            hostname=settings.SMTP_HOST,
            port=int(settings.SMTP_PORT),
            username=settings.SMTP_USER,
            password=settings.SMTP_PASSWORD,
            use_tls=True,          # 포트 465 SSL 직접 연결
            timeout=15,
        )
        logger.info(f"OTP 메일 발송 완료 → {email}")

    except aiosmtplib.SMTPAuthenticationError as e:
        logger.error(f"SMTP 인증 실패: {e}")
        raise RuntimeError("이메일 인증 실패: 하이웍스 계정 정보를 확인하세요.")
    except aiosmtplib.SMTPConnectError as e:
        logger.error(f"SMTP 연결 실패: {e}")
        raise RuntimeError("메일 서버 연결 실패: smtps.hiworks.com:465 접근을 확인하세요.")
    except Exception as e:
        logger.error(f"메일 발송 오류: {e}")
        raise RuntimeError(f"메일 발송 중 오류 발생: {e}")
```

## 4. `app/routers/auth.py` — `await` 키워드 추가

`send_otp_email`이 `async` 함수로 바뀌었으므로, 호출부에 `await`을 추가한다.

**변경 전**

```python
# ❌ 기존 — await 없음
send_otp_email(email=user.email, otp=otp, name=user.name)
```

**변경 후**

```python
# ✅ 수정 — await 추가
await send_otp_email(email=user.email, otp=otp, name=user.name)
```

> 호출하는 라우터 함수 자체도 반드시 `async def`여야 한다.  
> 이미 `async def`로 선언되어 있다면 `await` 추가만으로 완료된다.

```python
# routers/auth.py 호출 컨텍스트 예시
@router.post("/auth/step1")
async def request_otp(payload: OTPRequestBody):   # ← async def 확인
    user = user_repo.get_user_by_emp_id(payload.emp_id)
    otp  = generate_and_store_otp(payload.emp_id)
    await send_otp_email(email=user.email, otp=otp, name=user.name)  # ← await
    return {"message": "인증 코드가 발송되었습니다."}
```

## 5. 발송 테스트 (CLI 단독 실행)

```bash
python - << 'EOF'
import asyncio, ssl, os
from dotenv import load_dotenv
from email.mime.text import MIMEText
import aiosmtplib

load_dotenv()

async def test():
    to_ = input("수신 이메일 주소 입력: ")
    msg = MIMEText("HiWorks aiosmtplib 연결 테스트", "plain", "utf-8")
    msg["Subject"] = "[테스트] OTP 발송 확인"
    msg["From"] = os.getenv("SMTP_FROM")
    msg["To"]   = to_

    await aiosmtplib.send(
        msg,
        hostname=os.getenv("SMTP_HOST"),
        port=int(os.getenv("SMTP_PORT")),
        username=os.getenv("SMTP_USER"),
        password=os.getenv("SMTP_PASSWORD"),
        use_tls=True,
        timeout=15,
    )
    print("✅ 발송 성공")

asyncio.run(test())
EOF
```
